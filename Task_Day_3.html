Task 3: Steps to Add an Image in HTML

Steps to add an image in HTML

  1. It is necessary to get a photo of the **bike**.
  2. If there is no photo, take it from a browser or stock photo site.
  3. Copy the image address (URL).
  4. Paste the image address in the **img** tag's **src** attribute.
  5. If the photo doesn't show, add an **alt** attribute (alternate text) for accessibility.

Task 4: Common Image Formats

JPEG
Compressed Image Format, great for **photographs** of bikes (Joint Photographic Expert Groups)
PNG
High Quality Image Format, supports transparency, good for **bike logos** (Portable Network Graphics)
SVG
Vector Image Format Made Of Paths, not pixels, perfect for **scalable graphics** (Scalar Vector Graphics)
